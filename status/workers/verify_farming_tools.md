## Summary

- **9 of 11 actions fully work end-to-end** (YES)
- **2 actions have issues** (PARTIAL):
  - **#7 Tool Switching**: Keys 1–5 select the inventory hotbar slot, not the equipped tool (`PlayerState.equipped_tool`). The actual tool (Hoe/Axe/etc.) cycles only via Q/E. The HUD correctly reflects both, but the 1–5 → equip-tool path does not exist.
  - **#11 Scythe on Grass**: Works mechanically (Weed entities are cut, fiber dropped, entity despawned), but hay is never produced by the scythe. Hay is purchaseable from shops only.
